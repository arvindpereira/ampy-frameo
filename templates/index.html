    First-Time Setup Guide

    1. On the frame, go to Settings → About.
    2. Toggle Beta Program to ON.
    3. Toggle the new ADB Access option to ON.
    4. Connect the frame to your Mac with a USB data cable.
    5. Click the button below to enable wireless mode on port 5555:
    6. Look at the frame screen, allow debugging, and select "Always allow".
    7. Unplug the USB and connect wirelessly using the IP!
